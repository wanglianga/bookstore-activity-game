# 书店活动经营游戏

一款基于 PixiJS 的书店经营模拟游戏，玩家需要经营一家书店，通过安排活动、进货管理、人员配置来提升销售额和会员活跃度。

## 原始需求

> 做一个书店活动经营游戏，PixiJS 场景展示书架、咖啡区、活动区、收银台和会员服务台。玩家每天安排主题分享、作者见面、孩子体验或新书分享，同时进货图书、布置座位、安排店员和准备活动物料。顾客有购书偏好、停留时间、会员等级和活动兴趣，活动能带来人气但会占用卖场空间。玩家目标是在租金压力下提升销售额、会员活跃和活动口碑。

## 启动方式

### 前置要求

- Node.js >= 16.0.0
- npm 或 pnpm 包管理器

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

访问地址：http://localhost:3000

#### 3. 构建生产版本

```bash
npm run build
```

#### 4. 预览生产版本

```bash
npm run preview
```

## Docker 启动方式

### 前置要求

- Docker
- Docker Compose

### 一键启动

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

停止服务：

```bash
docker compose down
```

访问地址：http://localhost:3000

## 游戏玩法

### 基本操作

1. 点击 **"开始营业"** 开始新的一天
2. 每天营业时间为 9:00 - 21:00
3. 顾客会自动进入书店浏览和购买
4. 一天结束后会显示经营数据和净利润

### 活动系统

每天可以举办一种活动：

- **主题分享会** - 费用 ¥300，人气+15，空间占用 20%
- **作者见面会** - 费用 ¥800，人气+30，空间占用 35%，大幅提升销量
- **儿童体验活动** - 费用 ¥200，人气+10，空间占用 25%，吸引家庭客户
- **新书发布会** - 费用 ¥400，人气+20，空间占用 15%，刺激消费

### 经营管理

- **进货** - 购买各类图书补充库存，库存不足会影响销售
- **人员** - 雇佣/解雇员工，员工越多服务越好，但日薪支出也越高
- **座位** - 增加/减少座位，座位越多顾客停留越久

### 顾客系统

顾客有不同类型：
- 普通顾客 - 购买意愿一般
- 资深读者 - 购买意愿高，停留久
- 学生顾客 - 消费较低
- 白领精英 - 消费较高
- 家庭顾客 - 消费高，易被儿童活动吸引

### 会员系统

满意的顾客可能会成为会员：
- 铜牌会员 - 95折
- 银牌会员 - 9折
- 金牌会员 - 85折
- 钻石会员 - 8折

## 技术栈

- **PixiJS v7** - 2D 渲染引擎
- **Vite** - 构建工具
- **Vanilla JavaScript** - 原生 JavaScript 实现
- **CSS** - 样式
